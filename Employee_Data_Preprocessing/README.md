# Netflix Data Cleaning and Preprocessing Pipeline

## Project Overview

This project demonstrates a complete data cleaning and preprocessing workflow using the **Netflix Movies and TV Shows Dataset**.

The primary objective was to identify and resolve common data quality issues, making the dataset suitable for data analysis and future machine learning applications.

---

## Objectives

- Load the raw dataset
- Perform initial data inspection
- Analyze data quality
- Handle missing values
- Validate data types
- Perform feature engineering
- Standardize textual data
- Export a cleaned dataset

---
